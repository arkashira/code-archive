# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $0.10/user/month
- **Storage**: $0.05/user/month (assuming average usage of 10GB per user)
- **Bandwidth**: $0.02/user/month (assuming average data transfer of 5GB per user)
  
**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.02 = $0.17/user/month**

### Pricing Tiers
1. **Basic Tier**: $5/month
   - Features: 50GB storage, Basic backup & restore, Email support
2. **Pro Tier**: $15/month
   - Features: 200GB storage, Advanced backup & restore, Integration with 3 development tools, Priority email support
3. **Enterprise Tier**: $30/month
   - Features: 1TB storage, Custom backup & restore, Integration with unlimited development tools, Dedicated account manager, 24/7 support

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: $30 - $50 per user (including marketing, sales, and support costs)

### Lifetime Value (LTV) Estimate
- Average Revenue per User (ARPU) for each tier:
  - Basic: $5/month
  - Pro: $15/month
  - Enterprise: $30/month

- Average customer lifespan: 24 months

**LTV Calculation**:
- Basic: $5 * 24 = $120
- Pro: $15 * 24 = $360
- Enterprise: $30 * 24 = $720

### Break-even Users Count
- Fixed Costs (monthly): $1,000 (estimated operational costs)
- Contribution Margin per User:
  - Basic: $5 - $0.17 = $4.83
  - Pro: $15 - $0.17 = $14.83
  - Enterprise: $30 - $0.17 = $29.83

**Break-even Users Count**:
- Basic: $1,000 / $4.83 ≈ 207 users
- Pro: $1,000 / $14.83 ≈ 67 users
- Enterprise: $1,000 / $29.83 ≈ 34 users

### Path to $10K MRR
To achieve $10,000 Monthly Recurring Revenue (MRR), we can calculate the number of users needed for each tier:

1. **Basic Tier**: 
   - $10,000 / $5 = 2,000 users

2. **Pro Tier**: 
   - $10,000 / $15 ≈ 667 users

3. **Enterprise Tier**: 
   - $10,000 / $30 ≈ 334 users

**Summary**: 
- To reach $10K MRR, we can target:
  - 2,000 Basic users
  - 667 Pro users
  - 334 Enterprise users

This analysis provides a clear financial framework for the code-archive product, ensuring that we can strategically plan our marketing and sales efforts to achieve profitability.
